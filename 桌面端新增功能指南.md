# ChatSystem 桌面端新增功能指南

## 新增功能概述

根据考核要求，桌面端现在补充了网页端的所有功能，包括：

### 1. 好友管理功能
- **搜索用户**：按用户名或昵称搜索其他用户
- **发送好友申请**：向可添加的用户发送好友申请
- **删除好友**：从好友列表中删除好友关系

### 2. 历史消息功能
- **消息查询**：按好友、关键词、日期范围筛选历史消息
- **文件下载**：支持下载历史消息中的文件
- **删除记录**：删除个人历史消息记录（仅对自己不可见）

## 使用方法

### 好友管理
1. 登录桌面客户端
2. 点击右上角的"好友管理"按钮
3. 在搜索框中输入用户名或昵称搜索用户
4. 点击"发送申请"按钮向用户发送好友申请
5. 在"我的好友"区域可以查看和管理当前好友

### 历史消息
1. 登录桌面客户端
2. 点击右上角的"历史消息"按钮
3. 选择好友（或"全部好友"）、输入关键词、设置日期范围
4. 点击"查询"按钮查看符合条件的消息
5. 支持下载文件和删除消息记录

## API接口说明

### 好友管理相关API
- `GET /api/friends/search?keyword={keyword}` - 搜索用户
- `POST /api/friends/requests` - 发送好友申请
- `DELETE /api/friends/{friendId}` - 删除好友

### 历史消息相关API
- `GET /api/messages/history?friendId=&keyword=&startDate=&endDate=` - 查询历史消息
- `DELETE /api/messages/history/{messageId}` - 删除历史消息

## 技术实现

### 后端改动
1. **FriendsController**：新增搜索用户、发送申请、删除好友接口
2. **MessagesController**：新增历史消息查询和删除接口

### 前端改动
1. **FriendManagementWindow**：好友管理界面
2. **HistoryWindow**：历史消息界面
3. **DesktopApiClient**：新增API调用方法
4. **MainWindow**：添加功能入口按钮

## 注意事项

1. **登录状态**：使用功能前需确保已成功登录
2. **权限控制**：所有接口都需要DesktopToken认证
3. **数据一致性**：删除操作仅对当前用户生效，不影响其他用户
4. **系统兼容性**：支持Windows平台，需要.NET 8.0运行时

## 后续优化建议

1. **性能优化**：为大量消息数据添加分页功能
2. **用户体验**：添加搜索过程中的加载状态提示
3. **功能扩展**：考虑添加消息导入/导出功能
